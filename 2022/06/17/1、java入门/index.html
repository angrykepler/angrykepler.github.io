7、数组及初始化

数组初始换总共有三种：静态初始化、动态初始化、默认初始化

1.静态初始化

除了用new关键字来产生数组之外，还可直接在定义的数组的同时为数组元素分配空间并赋值

例如：

int[] arr = {12,23,56};

int[] arr = new int[]{}{12,23,56};

注意:

\1. new int[]{}{12,23,56};错误

\2. int[] arr;

​ arr = {12,23,56};错误

2.动态初始化

数组定义与数组元素分配空间并赋值的操作分开进行

例如：

int[] arr;

arr = new int[3]

arr[0] = 12;

arr[1] = 45;

3. 默认初始化

数组是引用类型
